# VirgoCards

Plataforma colaborativa de tarjetas didácticas con IA, pensada para que estudiantes creen, compartan y mejoren material de estudio entre generaciones.

## Stack tecnológico
- Backend: Laravel 12
- Frontend: React + Vite
- Base de datos: PostgreSQL
- Autenticación: Laravel Sanctum
- IA: OpenAI API / Gemini API

## Estructura del proyecto
- `backend/` — API REST en Laravel
- `frontend/` — SPA en React
- `docker/` — configuración de contenedores
- `docs/` — documentación y diagramas (visión, requisitos, casos de uso, base de datos, API, UI, roadmap)
